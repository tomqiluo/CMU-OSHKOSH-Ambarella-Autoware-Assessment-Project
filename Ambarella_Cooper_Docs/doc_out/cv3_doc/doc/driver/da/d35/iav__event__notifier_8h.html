typedef int(* iav_event_cb) (void *info, void *private_data)

@description: Common event callback for report different event info to private drivers use. 1: IAV_EVENT_NOTIFIER_EIS_RAW_INFO: called in ISR bottom half, cannot be blocked.

Parameters
infoevent info for private drivers, should be converted to different event info type accordingly. 1: IAV_EVENT_NOTIFIER_EIS_RAW_INFO: (struct iav_event_raw_info *)
private_datafor use by the owner of the @iav_event_cb.
Returns
: 0:success, failure: negative error code.
