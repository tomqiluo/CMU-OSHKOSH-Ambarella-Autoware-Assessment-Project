1. Sensor Module Parameters

Sensor drivers have some parameters that is mandatory or optional. For different system, those parameters can be different. Improper setting may cause the sensor(s) not working. For more examples on how to use those parameters, refer to ADVANCED - Multi-VIN & Multi-Channel
