cv_daddr_t result_index_daddr

The absolute physical address of matching index list. This list has CAVALRY_HARRIS_V_BLOCKS * CAVALRY_HARRIS_H_BLOCKS * CAVALRY_HARRIS_MAX_POINTS_PER_BLOCK entries if the FEX feature point list is provided as the target list. Each element is type uint16_t, and indicates the index of the matched feature point in the reference list.
