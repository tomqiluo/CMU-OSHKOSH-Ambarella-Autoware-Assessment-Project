This API queries the frame-based data descriptions. The frame-based data includes frame bit stream data, frame statistic data, frame raw data, frame YUV data, frame ME1 data, and bit stream info data for smart rate control library.

◆ IAV_IOC_QUERY_MEMBLOCK

#define IAV_IOC_QUERY_MEMBLOCK   IAVENC_IOWR(IOC_QUERY_MEMBLOCK, struct iav_querymem *)

This API queries the physical addresses for all the memory partitions shared between IAV and user application and the memory blocks of all canvases inside the DSP buffer. The start addresses, lengths, and data offsets of all memory partitions and canvases' YUV & ME are returned in information.

The application should further map the physical addresses into user space virtual addresses, before any operations on these buffers.
