◆ net_circle_cnt

IN uint32_t net_circle_cnt

Duplicate dag_cnt and dag_desc N times in struct run_dags. Networks's input and output should set no_mem flag. Networks's input and output size are "port * net_circle_cnt" and the address are continuous. It will save (N-1) ioctl interactive time between ARM and VP

◆ net_loop_cnt

IN uint32_t net_loop_cnt

Network loop run internal count. Blob memory size will be increased to "blob * net_loop_cnt". The port size is "port * net_loop_cnt" and the address is continuous. It will save (N-1) dvi loading time

◆ net_mem_total

The total network memory that is required. The app should allocate this size for the network. Every DVI and blob is aligned to 32-bit

◆ no_chip_check

Don't check chip arch between NET.bin and libnnctrl. 0: check chip arch; 1: don't check chip arch. It's debug proposal, shouldn't set this for production.

Examples
test_nnctrl.c, and test_nnctrl_mfd.c.

IN uint32_t separate_intermediate_io_mem

If seperate intermediate IO memory. Default is 0 If there are multi-networks, enable this can reduce the memory usage by share the same intermediate IO memory. Must satisfy two conditions when enable this: 1. No network preemption, all network use same priority;
