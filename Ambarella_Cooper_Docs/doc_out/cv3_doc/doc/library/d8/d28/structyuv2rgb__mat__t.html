Detailed Description

The yuv2rgb matrix parameters. The yuv2rgb_mat is defined as below [R G B]T = [yc 0 rv; yc -gu -gv; yc bu 0] x [Y-yb U-128 V-128]T.

Use below matrix to closely match OpenCV's result when call cv2.cvtColor(nv12, cv2.COLOR_YUV2RGB_NV12)

YUV2RGB_OpenCV: [yc, rv, gu, gv, bu, yb] = [1.164, 1.596, 0.391, 0.813, 2.018, 16].
