Detailed Description

The network of Faster-RCNN configuration in initialization.

The difference of Tensorflow between Caffe when enable is_tensorflow:

  1. RPN_cls equal "fg_cls - bg_cls" instread of "fg_cls in softmax" in library.
  2. RPN_bbox is feed by (y1, x1, y2, x2) bbox type. 3: ROI Pooling using CropAndResize and MaxPool2D.
  3. No backgourd bbox in RCNN.
