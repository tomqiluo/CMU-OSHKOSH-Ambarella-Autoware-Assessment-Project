◆ rescale_size

struct iav_window rescale_size

Rescale size for pyramid layer 1. The scale factor of W or H (layer0 -> layer1) must be in the range of 1/8X (Upscale) to 8X (Downscale), while the scale factor from layerN (1<= N < 6) to layerN+1 is fixed 2X (Downscale) in both width & height. Only valid when "scale_type == IAV_PYRAMID_SCALE_ARBITRARY".
