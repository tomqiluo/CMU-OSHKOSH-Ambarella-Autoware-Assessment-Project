◆ warp_area_behavior

u16 warp_area_behavior

Specify warp area behavior. 0 (WARP_AREA_CHANGE), is to change the selected warp area configuration on the fly. 1 (WARP_AREA_UPDATE). is to update all warp areas configuration at the same time. 2 (WARP_AREA_CLEAR), is to clear all warp areas configuration at the same time.
