◆ EA_IMG_RESOURCE_MAX_HOLDING_NUM

#define EA_IMG_RESOURCE_MAX_HOLDING_NUM   50

The limitation to hold data without drop.

Each data got from ea_img_resource_hold_data() should be dropped by ea_img_resource_drop_data(). If the drop is forgotten and the limitation number is reached, there will be a waring message in each call of ea_img_resource_hold_data().
