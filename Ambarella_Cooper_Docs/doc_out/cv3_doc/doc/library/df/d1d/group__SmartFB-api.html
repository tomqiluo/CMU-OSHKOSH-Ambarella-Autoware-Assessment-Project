◆ smartfb_draw_mosaic_with_argb8888()

AMBA_API int smartfb_draw_mosaic_with_argb8888 ( smartfb_mosaic_t *  fb_mosaic)

This API is used to draw mosaic with argb8888. Mosaics drawn through this api are true color format. A(alpha) uses 8bit, R(red) uses 8bit, G(green) uses 8bit, and B(blue) uses 8bit, so one pixel occupies 32-bit(four bytes).

Note: All interfaces use clut8bpp by default, except for some interfaces with "argb8888" suffix like this.

◆ smartfb_set_clut()

AMBA_API int smartfb_set_clut ( u16 *  r_table,
u16 *  g_table,
u16 *  b_table,
u16 *  trans_table 
)

This API is used to set the framebuffer's color lookup table.

Note: All the smartfb_draw_<xxx> interfaces based on clut8bpp are designed with the default color table, please refer to 2.3 CLUT for the default color table. If the user uses this interface to reset the color table, it may break the original design of some interfaces and cause unexpected problems.

Parameters
r_tableSpecify the address of lookup table for R(red) component.
g_tableSpecify the address of lookup table for G(green) component.
b_tableSpecify the address of lookup table for B(blue) component.
trans_tableSpecify the address of lookup table for A(alpha) component.
Returns
0: success; -1: error.
