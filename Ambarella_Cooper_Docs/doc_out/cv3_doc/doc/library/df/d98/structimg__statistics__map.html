Detailed Description

Description of AAA statistical data map All the addresses are allocated from user space memory. Driver will report AAA statistical data offset from IAV memory partition 'IAV_PART_DSP_FIFO'. User can mmap IAV_PART_DSP_FIFO partition in user space, and then can get the AAA statistical data through the offset.
