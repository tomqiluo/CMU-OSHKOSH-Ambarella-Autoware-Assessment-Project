Field Documentation

◆ buf_height

u16 buf_height

Buffer height. For single color mask: it is in pixel unit and equal to main buffer height. For multi color mask : it is downscaled from VIN's height.

◆ buf_pitch

u16 buf_pitch

Buffer pitch. For single color mask: it is aligned to 32 bytes(in byte unit). The unit depends on the privacy mask type. For multi color mask : it is downscaled from VIN's width.

◆ channel_id

u8 channel_id

Channel ID

◆ clear_user_mask

u8 clear_user_mask

Flag to clear user privacy mask memory only when PM is disabled (enable = 0). 0 (default) : Keep user PM memory as is, no clear. 1 : Clear user PM memory.
