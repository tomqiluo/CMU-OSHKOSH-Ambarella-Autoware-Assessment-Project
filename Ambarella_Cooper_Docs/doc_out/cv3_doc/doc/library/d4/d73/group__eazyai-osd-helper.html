enum ea_osd_default_16_colors_e

The enum definition of 16 colors.

Each color uses 16 index values which have the same color but different transparency. The indexes for each color is [color_id * 16, color_id * 16 + 15]. The alpha value for each color is in sequence of "0, 15, 31, 47, 63, 79, 95, 111, 127, 143, 159, 175, 191, 207, 223, 255". The formula to convert index [0, 14] to transparency is int(index * 255 / 16). The formula to convert a alpha value to index [0, 15] is int(alpha / 16).

Refer to https://www.december.com/html/spec/color16.html for the RGB values of 16 colors.
