AMBA_API const ea_corner_arrays_t * ea_orb_extractor_pyramid_corners ( ea_orb_extractor_t *  extractor)

Get the detected ORB corners in pyramid levels.

The positions of the corners are according to the pyramid level 0. The data will be valid until the next call of ea_orb_extractor_detect.

Parameters
extractorA pointer representing an ea_orb_extractor object.
Returns
const ea_corner_arrays_t * An array holding sub pyramid corner arrays. [n] is the array of pyramid level n.
