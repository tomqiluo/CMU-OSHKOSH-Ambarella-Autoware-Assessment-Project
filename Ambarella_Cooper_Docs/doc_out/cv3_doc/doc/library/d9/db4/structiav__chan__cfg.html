This is to specify vcap_mode for each channel. Only valid when chan_vcap_mode_enable = 1. Vcap modes for different cases.

See also
dsp_mode. 0x00: Advanced ISO mode. This is the mode for lowest latency from VIN to VOUT. 0x05: Line interleaved HDR mode. This is the mode for 3X HDR and multi-pass scale. 0x7F: Mode none.
