◆ ea_net_load()

AMBA_API int ea_net_load ( ea_net_t *  net,
ea_net_load_type_t  load_type,
void *  from,
int  max_batch 
)

Load a network from the cavalry binary file or the DRAM buffer.

Simulator platform: it loads a network from Ades cmd file and Ades folder if ades_cmd_file is set in ea_net_params_t structure. Otherwise it loads network from parser output folder for Acinf.

Note
On simulator platform, users need to make sure that the Ades file and the json file remains the original folder hierarchy with their parameter binaries files, which means that NEITHER the Ades command file NOR the parser json file itself CAN be regarded as an independant network, they must be used with their parameter binaries files.
Parameters
netA pointer which represents a network.
load_typeA type in ea_net_load_type_e.
fromChangeable meaning decided by load_type.

File path (const char *) if load_type is EA_NET_LOAD_FILE.

Buffer address (const uint8_t *) if load_type is EA_NET_LOAD_DRAM.

Parameters
max_batchThe maximum batch of the network in one forward.
Returns
int EA_SUCCESS: OK, EA_FAIL: Error
