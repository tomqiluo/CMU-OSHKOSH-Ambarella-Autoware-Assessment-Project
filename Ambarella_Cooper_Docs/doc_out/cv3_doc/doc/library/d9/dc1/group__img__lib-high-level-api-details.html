int  chan_id 
)

This API sets the HDR sensor shutter row and the AGC index group to the sensor driver for the specified VIN channel.

The sensor shutter and gain registers are updated at the VSYNC interrupt, in order to prevent an issue with the synchronization of the HDR sensor offsets.

In comparison to other sensors, it is difficult to define a range limitation for this API.

  • for Linear(or non-HDR) mode, there is only one shutter and AGC needed;
  • for HDR sensors, every long, middle, and short exposure should be specified using valid shutter values and AGC values./n This means that for HDR sensors, both a shutter group and an AGC group are needed.
    Parameters
    fd_iavsets the operation handler of IAV
    p_sensor_gpsets pointer of shutter and AGC group hdr_sensor_gp_t
    channel_idsets the specified Channel ID(starts from 0)
    Returns
    0 = success, -1 = error.
