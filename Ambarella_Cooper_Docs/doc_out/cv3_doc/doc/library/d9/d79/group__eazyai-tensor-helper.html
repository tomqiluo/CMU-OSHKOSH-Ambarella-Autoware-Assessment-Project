◆ ea_tensor_t

typedef struct ea_tensor_s ea_tensor_t

A type definition of the private ea_tensor_s structure.

The detailed definition of struct ea_tensor_s is hidden in ea_tensor.c. The operations on ea_tensor_s can be done by the API functions of ea_tensor without using the member variables directly.

The instance of ea_tensor_t is an object which represents a piece of memory holding a multidimensional array data. The multidimensional array data is also called a tensor. The memory of a tensor can come from the IAV interfaces, be allocated on IAV anonymous partition, be allocated on the cavalry memory, or be allocated on heap.

The memory from IAV and cavalry memory could be accessed among VP, DSP and CPU devices, so syncing cache though the API functions of ea_tensor should be considered if the CPU cache is enabled for the memory.
