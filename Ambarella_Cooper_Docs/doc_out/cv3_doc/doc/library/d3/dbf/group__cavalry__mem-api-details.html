◆ cavalry_mem_alloc()

AMBA_API int cavalry_mem_alloc ( IN unsigned long *  psize,
OUT unsigned long *  pphys,
OUT void **  pvirt,
IN uint8_t  cache_en 
)

This API allocates the memory from the CV user memory. If process have not free memory before exit, driver will auto recycle these leaked memory. Cache memory exist between ARM and DRAM. Turn on

Parameters
cache_encan boost ARM process speed.
psizethe pointer to total size want allocate
pphysthe pointer to physical address that return
pvirtthe pointer to virtual address that return
cache_enthe flag to enable cached memory. 0: non-cache; 1: cache
Returns
0 = success, -1 = error. -EBUSY if memory is busy, should retry alloc it.
Examples

This API allocates the memory from the CV user memory. If process have not free memory before exit, driver DO NOT auto recycle these leaked memory. Then it cause memory leakage issue. To avoid this issue, please use API cavalry_mem_alloc(). Cache memory exist between ARM and DRAM. Turn on

pphysthe pointer to physical address that return
pvirtthe pointer to virtual address that return
cache_enthe flag to enable cached memory. 0: non-cache; 1: cache
Returns
0 = success, -1 = error. -EBUSY if memory is busy, should retry alloc it.

◆ cavalry_mem_sync_cache()

AMBA_API int cavalry_mem_sync_cache ( IN unsigned long  size,
IN unsigned long  phys,
IN uint8_t  clean,
IN uint8_t  invalid 
)

This API syncs the cached memory when cache_en is set in 2.2 Allocate Memory. It will return an error if it is used with non-cached memory. Can sync one slice of total memory. At the begin need sync network's dvi memory after nnctrl_load_net, then most of time sync Network's Input/Ouput memory.

Parameters
sizesize of memory
physphysical address
virtvirtual address
cleanthe flag to clean cache. 0: turn off; 1: turn on. Program Flow: 1.ARM write -> 2.clean cache -> 3.VP read
invalidthe flag to invalid cache. 0: turn off; 1: turn on. Program Flow: 1.VP write -> 2.invalid cache -> 3.ARM read
Returns
0 = success, -1 = error.
Examples
test_cavalry_mem.c, test_nnctrl.c, test_pvanet.cpp, test_vproc_abs.c, test_vproc_bayer2bgr.cpp, test_vproc_cdist.cpp, test_vproc_cvfilter.cpp, test_vproc_edge_det.cpp, test_vproc_flatten.c, test_vproc_harris.cpp, test_vproc_harris_live.cpp, test_vproc_memcpy.c, test_vproc_memset.c, test_vproc_morph.cpp, test_vproc_optlk.cpp, test_vproc_render3d_live.cpp, and test_vproc_statistics.c.
