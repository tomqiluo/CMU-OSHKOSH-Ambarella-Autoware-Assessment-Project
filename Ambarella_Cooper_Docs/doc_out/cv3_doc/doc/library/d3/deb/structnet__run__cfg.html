◆ split_num_run

uint32_t split_num_run

Issued split number times ioctl to driver for each network. Used case: When there are two independent networks in two threads or process, one is large, another is small. Split large network to N part when issued ioctl, then small network can get a chance to run instead of to waiting for large network finished
