◆ WARP_INPUT_MODE

enum WARP_INPUT_MODE
Enumerator
WIM_MIN_BOUNDING_RECT 

0, Minimum bounding rectangle. DRAM bandwidth and raw padding wise. Need more vertical warp resource like vwarp blk height and wait lines for H+V pipeline; For pipeline H + Rotate + H, it consumes more warp padding width for second H Warp

WIM_ORIGINAL_INPUT 

1, Original VIN size or premain buffer size. Cost more DRAM bandwidth and raw padding. For H+V pipeline, vwarp block height or waitlines wise; for H+Rotate+H, warp padding width wise.

WIM_MIN_WIDTH_MAX_HEIGHT 

2, Minimum bounding rectangle width and original VIN height. For pipeline H+V; H+Rotate+H.
