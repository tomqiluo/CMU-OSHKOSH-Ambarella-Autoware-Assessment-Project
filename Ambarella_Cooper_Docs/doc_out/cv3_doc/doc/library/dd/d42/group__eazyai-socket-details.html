AMBA_API int ea_socket_read ( ea_socket_t *  link,
long  timeout_us,
int *  p_fd,
void **  p_buf,
size_t *  p_len 
)

Read or try to read data from anyone of the current connected other sides.

Used to read the first block of data of a frame sent by an other side. The following data blocks of the same frame should be read by ea_socket_read_by_fd.

Parameters
linkA pointer representing an ea_socket object.
timeout_us-1: blocking, 0: non-blocking, positive: timeout to wait. *p_buf will be invalid at next read.
p_fdOutput the file descriptor of the other side which sends the data.
p_bufOutput the memory holding the data. The data will be kept valid until the next ea_socket_read.
p_lenOutput the byte-size of the data.
Returns
int EA_SUCCESS: OK, EA_FAIL: Error
