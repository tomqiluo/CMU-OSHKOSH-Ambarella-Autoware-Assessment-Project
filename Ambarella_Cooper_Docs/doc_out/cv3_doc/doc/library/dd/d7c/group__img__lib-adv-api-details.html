◆ awb_flow_set_cali_thre()

int awb_flow_set_cali_thre ( u16 *  p_r_th,
u16 *  p_b_th,
u32  update_vin_chan 
)

This API sets the threshold for the AWB calibration for the specified VIN channel. The parameter p_r_th refers to the r gain threshold while p_b_th refers to the b gain threshold.

The AWB calibration will not start if the absolute difference between the original and referential r gain is greater than p_r_th, or the absolute difference between original b gain and referential b gain is greater than p_b_th.

Parameters
p_r_thThreshold of r_gain, range: [80, 200]
p_b_thThreshold of b_gain, range: [100, 250]
update_vin_channelsets the channels controlled, refer to UPDATE_VIN_CHAN_INDEX(vin_index)
Returns
0 = success, -1 = error.
