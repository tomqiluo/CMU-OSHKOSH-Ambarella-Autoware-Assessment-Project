BASIC - On-the-Fly Settings

1. On the fly settings

On-the-fly settings enable parameters to be changed independently during encoding. The following parameters can be configured on the fly for each stream:

  • Crop Offset
  • Frame Rate
  • Monochrome
  • Bit Rate
  • GOP Length
  • IDR Frequency or Intervals
  • Force IDR
  • QP limit
  • ROI Encoding Tools
