Docker User Specified Document

Docker is an open source, application-container engine that enables developers to package their applications and dependencies into a portable container, then publish or virtualize them to any Linux machine. CooperHome Overview

CooperHome is an Ambarella web application that runs on the Ambarella CV2x, CV5x, CV7x, and CV3 systems on chip (SoCs) using the Cooper software development kit (SDK). CooperHome provides overview information of the SoC and development board; monitors the status of the system; starts CPU tasks, digital signal processor (DSP) pipelines, and CVflowÂ® tasks with simple clicks; profiles the performance of various use cases; and captures output data.
