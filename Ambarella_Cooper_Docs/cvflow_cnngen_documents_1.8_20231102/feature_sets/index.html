CVflow CNNGen Developmen

The docuement of CVflow CNNGen Development provides detailed information of NN convert with CNNGen toolchain and NN deployment with Cooper SDK for the CVflow® Vector Processor (VP), a key vision processing module for executing neural networks efficiently in Ambarella’s family of computer vision (CV) chips.
