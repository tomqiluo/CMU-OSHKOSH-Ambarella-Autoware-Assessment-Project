Detailed Description

Analyzes the CV board output or the PC output for the segmentation evaluation.
    A visible statistic TXT file for humans is generated under the path given as the command line parameter.

Public attributes:
    net_out_bin_list (str): Network output bin files list.
    out_layer_shape (str): The output shape of the output layer.
    out_layer_data_format (str): The data format of output layer
    gt_list (str): List of ground truth images
    pad (int): padding alignment byte num, should be 0|32|64.
    gt_dataset (str): dataset of ground truth, like "voc_07" "cityscapes".
    seg_out_path (str): folder to save output segmentation result in image.
    seg_report_file (str): file to save output segmentation report.
    logger (str, optional): logger. Defaults to None.
    log_level (int, optional): log_level. Defaults to logging.DEBUG.
