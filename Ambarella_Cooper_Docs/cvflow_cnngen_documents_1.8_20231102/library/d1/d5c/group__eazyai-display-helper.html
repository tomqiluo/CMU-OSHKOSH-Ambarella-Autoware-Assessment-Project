#define EA_DISPLAY_STREAM_SERVER_TCP_PORT_BASE   14100

The base of the TCP server port number for the display type of EA_DISPLAY_STREAM_SERVER and EA_DISPLAY_STREAM_CLIENT.

The port base + stream_id will be in use between the server and the client after ea_display_new() is called.
