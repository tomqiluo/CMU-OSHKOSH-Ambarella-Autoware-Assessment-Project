Detailed Description

Class represents inference with dummy input.
Public attributes:
    - set_times: Set iteration times
    - set_inference_model_path: Set model path.
    - set_evk_work_dir: Set work dir in EVK
    - set_log_level: Set log level
    - run: Run inference with block
    - run_non_block: Run inference without block
