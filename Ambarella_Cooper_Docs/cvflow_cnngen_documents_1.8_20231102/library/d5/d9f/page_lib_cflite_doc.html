1 Introduction

This section provides information on the CVflow lite (CFlite), it includes a series of modules and tools based on Python which can help user to convert and deploy their networks easily.

  1. With unify interface, users can share the same interface of different methods on different chips with different CNNGen toolchain.
  2. With dummy convert, users can convert a network very quickly and get its performance with the model self, no need to know its detailed model information and config it.
  3. With simple convert flow, users can convert a network only with a YAML configuration file.
  4. With strong inference design, users can call python tools to inference a network on build server with CVflow chip, Ades, Acinference, and original framework easily.
  5. With clear structure package and tools, users can find different needed modules easily, and reduce learning cost.

2 CFlite Modules

CFlite has multiple fentures which can make users easily convert and deploy different networks.

Figure 2. CFlite Features.

For detailed API usage, please refer to 1 EazyAI CVflow Lite Python Library.

This library provides a series of below different python API modules.

  1. EazyaAI Generator, which provides python APIs to call CNNGen toolchain to convert the networks with a YAML configuration file, also provides dummy convert module for quick performance evaluation.
    Figure 2-1. CFlite Generator.
  2. EazyAI Inference Engine, which provides python APIs to inference the networks on CVflow chip, Ades and Acinference simulator, and Original framework.
    Figure 2-2. CFlite Inference.
  3. EazyAI System Modules, which provides python APIs to control the EVK board and setup video pipeline with Serial Port, Telnet, FTP, SSH, and even local sessions.
    Figure 2-3. CFlite Connection.
  4. EazyAI Utils, which for some basic usage.
    1. Graph Summary, which provides python APIs to get network basic information for Caffe, Tensorflow, and ONNX, such as input, output, density, and so on.
    2. Accuracy Evaluation, which provides python APIs to evaluate accuracy, such as mAP, TOPn, Cosine Similarity, and mIOU.

3 CFlite Python Tools

There are some python tools in this library that can be used.

Build $ eazyai_helper --show_tools
# CFLite Convert and Inference Tools:
1. [eazyai_helper]
- Show supported tools and utils in CFLite.
2. [eazyai_cfg]
- Create the configuration file of network for eazyai_cvt.
3. [eazyai_cvt]
- Convert the network to Ambarella format, and generate summary file for easy inference with eazyai_inf.
4. [eazyai_video]
- Enable CVflow and set up the video pipeline on EVK for EazyAI inference applications.
5. [eazyai_inf_simple_dummy]
- Run inference with dummy input for quick performance test.
6. [eazyai_inf]
- Run network forward inference with different platforms.
7. [eazyai_inf_simple_live]
- Run live demos on EVK, which needs to use eazyai_video to set video pipeline first.
8. [eazyai_profiler]
- This tool will generate a spreadsheet to describe CVflow hardware efficiency for current network.
# CFLite Utils:
1. [tf_remove_node]
- Remove specified nodes from Tensorflow models.
2. [im_reader]
- Read RGB file to image.
3. [simple_bincmp]
- Compare automatically binary files generated by ades and cavalry.
4. [mtcnn_get_pnet_scales]
- Calculate mtcnn pnet scales by original image resolution, minimum face size on the image and
a scaling factor.
5. [padding_append]
- Add padding to non-padding binary.
6. [convert_jpg2nv12]
- Convert the image dataformat from JPG to NV12.
7. [im_quant]
- Do image quant.
8. [binrotate]
- Rotate binary file with specific degree
9. [compare_classification_network_result]
- Compare and draw the classification network result.
