Detailed Description

Represents a CNNGen layercompare in convert process

Public attributes:
    - parser (class object): Inherited 'Parser()' object to obtain the Parser results.
    - vas (class object, optional): Inherited 'VAS()' object to obtain the VAS results. Defaults to None.
                                In CVB(CVFlowBackend) mode, this option is not required.
    - global_env (str, optional): Change the env file of layer compare. Defaults to None.
    - logger (str, optional): logger. Defaults to None.
    - log_level (int, optional): log_level. Defaults to logging.INFO.
    - set_model_path: Set the path of model
    - set_model_name: Set the name of model
    - set_caffe_prototxt_model_path: Set the path of caffe prototxt file
    - set_parser_output_path: Set the path of parser output
    - set_ambapb_file_path: Set amba pb file path, only for CVBackend.
    - set_layer_compare_output_path: Set the path of layer_compare
    - set_input_node_params: Set the node parameters of input node.
