Detailed Description

Calculates the cosine distance between the PC output and the CV board output.
    A visible statistic CSV file for human is generated under the path given as the command line parameter.

Public attributes:
    pc_bin_path (str): pc result bin folder or file path
    cavalry_path (str): cavalry result bin folder or file path
    pad (int): padding alignment byte num, should be 0|32|64.
    output_path (str): result file path
    data_type_pc (str): data type for pc bin
    data_type_cv (str): data type for cv bin
    channel_num (int, optional): channel number.  Defaults to None.
    height (int, optional): height. Defaults to None.
    width (int, optional): width. Defaults to None.
    logger (str, optional): _description_. Defaults to None.
    log_level (int, optional): _description_. Defaults to logging.DEBUG.
