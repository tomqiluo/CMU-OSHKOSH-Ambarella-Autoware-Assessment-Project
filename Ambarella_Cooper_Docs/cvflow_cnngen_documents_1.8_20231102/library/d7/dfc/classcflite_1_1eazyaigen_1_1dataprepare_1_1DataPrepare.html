Detailed Description

Represent a data prepare module which generates binary files and a binary file path list in a text file from a in_path
for one network input.
The in_path could be an image folder, a binary file folder or a text file containing a list of file paths.

Public attributes:
    - set_arguments: Set the arguments of in_path, in_file_ext, out_shape, out_data_format and transforms.
    - run: Do transformation and return a list of binary file paths in a text file.
    - builtin_transforms: Get information of builtin transforms.
