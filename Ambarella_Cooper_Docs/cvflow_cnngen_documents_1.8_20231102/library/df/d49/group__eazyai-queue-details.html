◆ ea_queue_new()

AMBA_API ea_queue_t* ea_queue_new ( )

Create a EA queue object.

A thread-safe queue with management of free carriers. The caller provides the free carriers at the beginning. Then the sender requests free carriers to hold new data before doing enqueue, and the receiver releases carriers after dequeue. The states of a carrier are init[add]->free[request]<->[release]requested[enqueue]<->[force request]enqueued[dequeue]->dequeued[release]->free.

Returns
ea_queue_t * A pointer which represents the queue object. NULL will be returned if failed.

◆ ea_queue_request_carrier_force()

AMBA_API void* ea_queue_request_carrier_force ( ea_queue_t *  queue,
void **  forced_out 
)

Request a free carrier by occasionally taking the enqueued carrier on the head.

If there is free carriers, a free carrier will be returned and its state changes from free to requested. If there is no free carrier, the enqueued carrier on the head will be force out to requested state and be returned.

Parameters
queueA pointer which represents the queue object.
forced_outA pointer which stores the pointer of the forced-out enqueued carrier.
Returns
A pointer which represent the carrier in requested state. NULL will be returned if there is no free and enqueued carrier.
