Detailed Description

Represents a CNNGen profiler tool for debug

Public attributes:
    - set_vas_base_name: Set the name of model, which should be the same with the .vas file
    - set_vas_output_path: Set VAS output folder path
    - set_cavalry_bin_path: Set cavalry binary path
    - set_output_path: Set the output path
    - set_evk_work_dir: Set the work dir on EVK, default it /tmp/
    - set_input_node_and_bin: Set the name and binary path of input nodes
    - run: run profiler in subprocess
    - set_env_path: Change the env file of CNNGen toolchain
