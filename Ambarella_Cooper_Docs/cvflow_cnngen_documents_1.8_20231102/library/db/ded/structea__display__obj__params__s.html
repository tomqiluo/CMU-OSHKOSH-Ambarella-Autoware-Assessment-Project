◆ blur_is_blocky

int blur_is_blocky

The DSP blur blocky flag. 0: not blocky; 1: blocky. It only works on CV5 by now.

◆ blur_strength

int blur_strength

The DSP blur strength for bbox. The larger the more heavily blurred. It should be >= ::MIN_BLUR_STRENGTH and <= ::MAX_BLUR_STRENGTH which are defined in iav_ioctl.h, but -1 can be used to disable the blur.
