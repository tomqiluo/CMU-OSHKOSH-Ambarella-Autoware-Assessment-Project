◆ disable_output_fp32_from_fp16_on_non_fp32_arch

int disable_output_fp32_from_fp16_on_non_fp32_arch

Disable output float32 data from float16 data on non-float32 CVflow ARCH like CV72. If this flag is not set, the CVflow ARCH has no float32 inside (like CV72), and the network's output is float16, ea_net_output() will return a float32 tensor which is converted from the network's float16 output. If this flag is set, ea_net_output() will return a float16 tensor if the network's output is float16. When the CVflow ARCH is Gen-2 (CV2, CV2x or CV5x), which has float32 inside, this flag is ignored and ea_net_output() will always return the network's original output.

◆ nvp_affinity

uint32_t nvp_affinity

The bitmap to set NVP affinity for multi-core CVflow platform. Choose which NVP(s) to use by set bit0: NVP0, bit1: NVP1, bit2: NVP2, ... If none NVP is chosen, the network will run on a NVP automatically. If only one NVP is chosen, the network will always run on this NVP. If multi NVPs are chosen, the network will run on one of these NVPs when it is idle.
