Create a color table image object by doing color quantization on an image from a .png or .jpg file.

Currently, OpenCV kmeans is used for color quantization.

Parameters

widthWidth of the color table image to be created.
heightHeight of the color table image to be created.
start_indexThe index value taking the first color in the RGB color table.
color_numNumber of colors to be clustered in color quantization.
Returns

AMBA_API int ea_soft_nms ( float *  x1y1x2y2score,
void *  aux,
size_t  aux_element_size,
int  num,
float  iou_threshold,
int  use_iou_min,
int  top_k,
ea_soft_nms_method_t  method,
float  discard_threshold,
float  sigma,
float *  out_x1y1x2y2score,
void *  out_aux,
int *  out_num 
)

soft-NMS. It does not directly delete the bbox with IOU greater than the iou_threshold, but reduces its classification score to alleviate the missed detection.

Parameters
x1y1x2y2scoreArray of which each item has a group of x1, y1, x2, y2, and score.
auxAnother data array.
aux_element_sizeSize of each item in aux.
numNumber of items in array.
iou_thresholdThreadhold of IOU to identify valid data item. Similar to threshold in ea_nms().
use_iou_minThe flag to use IOU over min, or else IOU over union will be used.
top_kKeep top k items as the valid result.
methodThe method to set the weight. Please refer to ea_soft_nms_method_t.
discard_thresholdAfter iteration, the bbox will be discarded if the bbox's score is less than it.
sigmaIt is used as a parameter of the Gaussian formula to set the weight, and it is only valid when the method is EA_SOFT_NMS_METHOD_GAUSSIAN. For other methods, please set it to 0.
out_x1y1x2y2scoreChoosen data from array x1y1x2y2score and sorted in score order.
out_auxChoosen data from array aux and sorted in score order.
out_numNumber of valid items in the choosen data.
Returns
int EA_SUCCESS: OK, EA_FAIL: Error
