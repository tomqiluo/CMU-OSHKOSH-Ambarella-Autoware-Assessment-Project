Detailed Description

Represents a CNNGen Cavalry in convert process

Public attributes:
    - vas (class object): Inherited 'VAS()' object to obtain the VAS results.
    - global_env (str, optional): Change the env file of cavalry. Defaults to None.
    - logger (str, optional): logger. Defaults to None.
    - log_level (int, optional): log_level. Defaults to logging.INFO.
    - set_model_input_nodes: Set the model input nodes.
    - set_cvb_json_file: Set cvb json file path, only for CVBackend.
    - set_vas_output_path: Set the path of vas output.
    - set_cavalry_output_path: Set the path of cavalry output.
    - set_cavalry_bin_name: Set cavalry binary name.
    - set_model_name: Set model name.
    - set_ades_output_path: Set the path of ades output.
    - generate_commands: Generate the cavalrygen commands
    - run: run cavalrygen in subprocess
