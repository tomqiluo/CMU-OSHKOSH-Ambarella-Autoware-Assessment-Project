Detailed Description

Represents a CNNGen parser in convert process

Public attributes:
    - set_work_dir: Set the work dir
    - set_model: Set the file path and name of model
    - set_caffe_prototxt_model_path: Set the prototxt model file path for caffe
    - set_output_path: Set the path of parser output
    - set_sideband: Set path of side band file
    - set_customer_node_parser: Set path of customer_node_parser
    - set_customer_node_dll: Set path of customer_node_dll
    - set_input_node_params: Set params of input node
    - set_output_node_params: Set params of output node
    - generate_commands: Generate the parser commands
    - run: run parser in subprocess
    - run_with_dummy_input: run parser with dummy DRA files in subprocess
    - set_env_path: Change the env file of CNNGen toolchain
