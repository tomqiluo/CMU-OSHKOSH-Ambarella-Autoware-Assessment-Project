Detailed Description

Analyzes the CV board output or the PC output for the mAP evaluation.
    A visible statistic CSV file for human is generated under the path given as the command line parameter.

Public attributes:
    label_num (int): total num of labels
    det_result (str): input file of network deploy result in uniformed format
    ground_truth (int): input file of ground truth in uniformed format
    dataset (str): dataset of ground truth. for example 'voc_07' 'voc_12' 'coco_2014'
    outfile (str): Output mAP result file path.
    norm (int): whether uniformed input data is normalized. use normalized data: --norm 1
    eval_iou_th (float, optional): iou threshold for evaluation. could be more than one value (seperated by space). Defaults to 0.5.
    debug (int, optional): debug flag. --debug 1. Defaults to 0.
    debug_label (int, optional): debug label. --debug 6. Defaults to 0.
    logger (str, optional): logger. Defaults to None.
    log_level (int, optional): log_level. Defaults to logging.DEBUG.
