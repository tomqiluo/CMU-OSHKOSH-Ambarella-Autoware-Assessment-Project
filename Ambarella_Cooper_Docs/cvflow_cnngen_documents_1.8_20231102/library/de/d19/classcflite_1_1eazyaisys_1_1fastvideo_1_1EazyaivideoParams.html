Detailed Description

Class EazyaivideoParams restores parameters of eazyai_video.sh

Public attributes:
    - evk_ip: EVK IP
    - camera_name: The name of the camera
    - camera_in: The resolution for camera
    - buffer_1: The resolution of the first VCA buffer
    - buffer_2: The resolution of the second VCA buffer
    - buffer_3: The resolution of the third VCA buffer
    - hdmi_1: The resolution of HDMI
    - stream_1: The resolution of stream_1
    - stream_2: The resolution of stream_2
    - manual_feed: Enable manual feed
    - enable_pyramid: Enable pyramid
    - blur: Enable blur
    - idle_state: Set EVK to idle state
    - show_supported_board: Show all supported board
