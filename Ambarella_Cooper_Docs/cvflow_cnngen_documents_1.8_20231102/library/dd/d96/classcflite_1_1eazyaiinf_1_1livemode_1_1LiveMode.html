Detailed Description

Class represents inference with IAV input.
Public attributes:
    - set_canvas_id: Set Canvas Id
    - set_nn_in_cf: Set model input color format
    - set_evk_work_dir: Set work dir in EVK
    - set_log_level: Set log level
    - set_pyramid: Set Pyramid Id
    - set_roi: Set ROI of image
    - set_hold_image: Set whether enable hold bgr image
    - set_stream_id: Set Stream Id
    - set_fsync_off: Set whether disable frame sync
    - set_stream_offset: Set overlay buffer offset for multi-stream display
    - set_vout_id: Set VOUT Id
