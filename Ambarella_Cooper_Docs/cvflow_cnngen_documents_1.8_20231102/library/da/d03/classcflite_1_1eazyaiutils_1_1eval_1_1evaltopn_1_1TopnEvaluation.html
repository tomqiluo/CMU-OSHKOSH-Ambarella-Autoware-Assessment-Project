Detailed Description

Analyzes the CV board output or PC output for the topN evaluation.
    A visible statistic CSV file for humans is generated under the path given as the command line parameter.

Public attributes:
    bin_path (str): img_bin folder path
    pad (int): padding alignment byte num, should be 0|32|64.
    label_path (str): lable folder path
    output_path (str): result file path
    logger (str, optional): logger. Defaults to None.
    log_level (int, optional): log_level. Defaults to logging.DEBUG.
