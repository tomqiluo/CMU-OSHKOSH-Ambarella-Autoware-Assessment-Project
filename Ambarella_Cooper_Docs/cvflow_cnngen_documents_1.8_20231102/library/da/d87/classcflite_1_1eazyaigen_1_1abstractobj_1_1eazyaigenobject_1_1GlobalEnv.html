Detailed Description

Represents a data structure contains CNNGen toolchain related information

Public attributes:
    - log_level: log level, default is INFO
    - env_file_path: Path of CNNGen env file
    - env_folder_path: Folder of CNNGen env file
    - toolchain_version: CNNGen toolchain version
    - platform: Target platform of CNNGen toolchain
    - work_dir: Path of work dir
    - alia_name: Alia name, default is ""
    - show_info: Show brief information
