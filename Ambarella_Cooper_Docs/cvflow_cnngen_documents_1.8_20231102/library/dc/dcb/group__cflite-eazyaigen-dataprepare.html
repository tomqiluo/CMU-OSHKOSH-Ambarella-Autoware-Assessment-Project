Set specific arguments of the data prepare object.

Args:
    in_path (str): The path of an image folder, a binary file folder or a text file containing a list of file paths.
    out_shape (str): The shape of data in final binary files. e.g. "1,3,320,480".
    out_data_format (str): The format of data in final binary files. e.g. int8|uint8|int16|uint16|int32|uint32|float16|float32.
    transforms (list): A list of transforms. e.g. 'transforms': [
{
    'class': "GenImageList",
    'arguments': {
        'color': "RGb",
        'mean': "127.5,127.5,127.5",
        'std': "128.0,128.0,128.0"
    }
},
    ].
    The value of 'class' could be the name of a builtin transform like "GenImageList|UseOriginalFiles|ReadBinaryFiles|GenMfccFeature".
    Or it could be a Python statement to import the custom transform class, like "from eazyaigen.dataprepare import ReadBinaryFiles".
    The custom transform class should inherit BaseTransform and implement the member functions set_arguments(self, **kwargs) and run(self, in_file_path: str, out_file_path: str),
    and use the same parameter names of run().
    The value of 'arguments' is specific for the class, which will be passed to the class's object by it's set_arguments() member function.
