Detailed Description

Represents a CNNGen VAS in convert process

Public attributes:
    - parser (class object): Inherited 'Parser()' object to obtain the Parser results.
    - global_env (str, optional): Change the env file of VAS. Defaults to None.
    - logger (str, optional): logger. Defaults to None.
    - log_level (int, optional): log_level. Defaults to logging.INFO.
    - set_vas_output_path: Set the path of vas output
    - generate_commands: Generate the vas commands
    - run: run vas in subprocess
